Reasoning split

The experiment treats route choice as a learned proposal problem. Syntax checks, Tau runs, route certificates, and replay verification remain deterministic gates. That split is the core claim readers can inspect.

LLM

Explains the workbench and can help translate user intent into structured work packets.

TauEnergy

Ranks route candidates using measured formula-shape evidence.

WES

Explores nearby candidates and records replayable traces for accepted checks.

Tau

Checks syntax, formulas, or certificates before any result is treated as valid.

Ordered-BDD curriculum

The curriculum starts from a broad base, then adds BDD-specific examples. The rise is small but useful: it shows fragment-specific examples can steer route ranking.

Assumptions

Assumption A: the public JSON snapshot accurately summarizes the local replay artifacts that generated it. Assumption B: Tau syntax and route behavior can drift, so current claims require fresh local replay before they are reused as current performance evidence.

Non-claims

This experiment does not claim that TauEnergy proves correctness, that TauJEPA proves future safety, or that a learned route selector should replace Tau's authoritative checks. The trained model is an ordering aid.
